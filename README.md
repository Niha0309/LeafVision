# LeafVision: Potato Leaf Disease Detection

## Overview

LeafVision is a deep learning framework for Potato Leaf Disease Detection using a hybrid **ResNet50 + U-Net inspired architecture**. The project classifies potato leaf diseases from image data using TensorFlow and Keras.

## Features

* Potato leaf disease classification
* ResNet50 feature extraction
* U-Net style architecture
* Data augmentation
* Confusion matrix visualization
* Model saving and prediction

## Technologies

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
* Scikit-learn

## Dataset Classes

* Bacteria
* Fungi
* Phytophthora
* Nematode
* Pests
* Virus
* Healthy
